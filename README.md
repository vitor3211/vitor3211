# 💫 Sobre mim:
Olá! Sou Vitor, desenvolvedor focado na construção de sistemas robustos e escaláveis. Minha paixão é o Backend, onde transformo regras de negócio em APIs eficientes e infraestruturas resilientes.

Sou um entusiasta da Matemática e utilizo o rigor analítico como ferramenta de precisão para otimizar algoritmos e estruturar arquiteturas complexas. Atualmente, foco na performance do ecossistema Java/Spring e na versatilidade do Python, onde aplico fundamentos estatísticos e algébricos para o processamento de dados e o estudo de Machine Learning na construção de modelos inteligentes.
